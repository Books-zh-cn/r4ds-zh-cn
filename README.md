# r4ds-zh-cn

## Description

Try to translate R for Data Science (the second edition) into Simplified Chinese.

-   **Original Book**: [r4ds](https://r4ds.hadley.nz/)

-   **Translation Book**: [r4ds-zh-cn](https://zhenghu159.github.io/r4ds-zh-cn/)
